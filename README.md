libobs-sharp
============

C# wrapper for libobs (part of obs-studio, multiplatform rewrite of Open Broadcaster Software)

### Requirements

* [obs-studio](https://github.com/jp9000/OBS-studio)
* .NET Framework 4.5 / Mono
